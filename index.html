API Endpoints Documentation

Available Endpoints:

  • GET /products - Retrieve a list of all products
  • GET /products/:id - Retrieve details of a specific product by its ID
  • POST /products - Add a new product
  • PUT /products/:id - Update an existing product by its ID
  • DELETE /products/:id - Remove a product by its ID
  • GET /products/search-by-name/:name - Search for products by name
  • GET /products/search-by-price/:price - Search for products by price
  • GET /products/search-by-id/:id - Search for products by their ObjectId
  • GET /products/search - Search for products by name and/or price
